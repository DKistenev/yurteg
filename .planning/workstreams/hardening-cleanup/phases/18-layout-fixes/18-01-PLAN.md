**1. Stats bar fix (LAY-01) — критический баг:**
Заменить блок строк 122-139. Текущий код создаёт `total_num`, `expiring_num`, `attention_num` до `with stats_row:`, поэтому NiceGUI добавляет их в DOM вне flex-контейнера. Результат: числа «прибиты к верху», вне строки.

Новый код — все labels создаются ВНУТРИ `with ... as stats_row:`:
```python
with ui.row().classes(STATS_BAR) as stats_row:
    with ui.column().classes(STATS_ITEM):
        total_num = ui.label("—").classes(STAT_NUMBER + " text-slate-900").props('aria-label="Всего документов"')
        ui.label("документов").classes(STAT_LABEL + " text-slate-500")
    ui.label("·").classes("text-slate-300 text-xl font-light self-center")
    with ui.column().classes(STATS_ITEM):
        expiring_num = ui.label("—").classes(STAT_NUMBER + " text-amber-600").props('aria-label="Документов истекает"')
        ui.label("истекают").classes(STAT_LABEL + " text-slate-500")
    ui.label("·").classes("text-slate-300 text-xl font-light self-center")
    with ui.column().classes(STATS_ITEM):
        attention_num = ui.label("—").classes(STAT_NUMBER + " text-red-600").props('aria-label="Требуют внимания"')
        ui.label("требуют внимания").classes(STAT_LABEL + " text-slate-500")
```
Переменные `total_num`, `expiring_num`, `attention_num` сохраняются для `_refresh_stats()` — функция вызывает `.set_text()` на них, оставить без изменений.

```python
with ui.row().classes(STATS_BAR.replace("bg-white", "bg-transparent")) as stats_row:
```
Это безопаснее — не трогает styles.py, который используется в других местах.

1. `floatingFilter: True` в column defs с `filter: "agTextColumnFilter"` — в AG Grid 31+ floatingFilter задаётся через `floatingFilterComponent`, не булеан. Если warnings связаны с этим, заменить `"floatingFilter": True` на `"floatingFilterComponentParams": {}` или убрать вообще.

2. Колонки с `"flex"` и одновременно `"width"` конфликтуют — убрать дублирование.

3. Проверить и добавить `suppressPropertyNamesCheck: True` в gridOptions если есть нераспознанные опции.

Найти в файле все `floatingFilter` вхождения. Если они есть — удалить их (они вызывают предупреждения в AG Grid Community 32+, floatingFilter для text filter теперь включён по умолчанию или задаётся иначе).
