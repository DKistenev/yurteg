# Комплексный аудит ЮрТэг — 2026-03-25

4 агента Opus, ~18 минут, line-by-line анализ всей кодовой базы.

## CRITICAL (7)

1. **INSERT OR REPLACE уничтожает пользовательские данные** — `modules/database.py:314-326`. При повторной обработке теряются review_status, lawyer_comment, manual_status, warning_days. Новый id ломает FK.
2. **5 полей из AI никогда не сохраняются в БД** — `is_template`, `payment_terms`, `payment_amount`, `payment_frequency`, `payment_direction` нет в schema.
3. **`db.get_contract()` не существует** — `registry.py:917`, `bulk_actions.py:62`. Метод называется `get_contract_by_id()`. Split panel крашится.
4. **«Скачать PDF» → 404** — `document.py:228-230`. Route `/download/{doc_id}` не существует.
5. **Настройка warning_days не читается** — сохраняется как `warning_days_threshold`, читается как `warning_days`.
6. **30+ `except Exception:` без логирования** — `document.py` (7 шт), `settings.py`, `registry.py`. Ошибки проглатываются.
7. **Foreign keys отключены** — `PRAGMA foreign_keys = ON` нигде нет. Orphaned rows.
