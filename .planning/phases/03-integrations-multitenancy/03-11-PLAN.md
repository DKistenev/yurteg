```python
# After showing all bindings, add confirm button
if _bind_summary["bindings"]:
    if st.button("Подтвердить привязку", key="confirm_bind_btn", type="primary"):
        from controller import move_record_to_client
        from modules.database import Database

        _src_db_path = client_manager.get_db_path(_selected_client)
        _moved = 0
        _move_errors = 0

        with Database(_src_db_path) as _from_db:
            # Get all records from source DB to find matching filenames
            _all_src = _from_db.get_all_results()
            _filename_to_id = {r["filename"]: r["id"] for r in _all_src}

            for _cl_name, _cl_files in _bind_summary["bindings"].items():
                _dest_db_path = client_manager.get_db_path(_cl_name)
                with Database(_dest_db_path) as _to_db:
                    for _fname in _cl_files:
                        _rid = _filename_to_id.get(_fname)
                        if _rid and move_record_to_client(_rid, _from_db, _to_db):
                            _moved += 1
                        else:
                            _move_errors += 1

        if _moved:
            st.success(f"Перемещено {_moved} документов")
        if _move_errors:
            st.warning(f"Не удалось переместить {_move_errors} документов")
        if _moved:
            st.rerun()
```

Key considerations:
1. The `_selected_client` variable (defined at line 566) is the CURRENT active client — this is the source DB from which records are moved
2. `_bind_summary["bindings"]` maps client_name → list[filename], we need record IDs from the source DB
3. `get_all_results()` returns list of dicts with "id" and "filename" keys
4. `move_record_to_client(record_id, from_db, to_db)` copies the row and deletes from source
5. After successful moves, `st.rerun()` refreshes the page to reflect changes
6. Both Database instances must be open simultaneously (from_db for read+delete, to_db for insert)
7. The button uses `type="primary"` for visibility and a unique key `confirm_bind_btn`
