       **llama-server singleton section (per D-09, D-10, D-11):**
       - Module-level `_llama_manager: LlamaServerManager | None = None`
       - `async def _start_llama():` — global _llama_manager, create Config(), check `config.active_provider != "ollama"` → return early, else create LlamaServerManager(port=config.llama_server_port), try: `await run.io_bound(manager.ensure_model)`, `await run.io_bound(manager.ensure_server_binary)`, `await run.io_bound(manager.start, get_grammar_path())`, set `_llama_manager = manager`. except: log warning with `logger.warning(f"llama-server не запустился: {e}")`.
       - `def _stop_llama():` — global _llama_manager, if _llama_manager and _llama_manager.is_running(): _llama_manager.stop(). Idempotent — safe to call multiple times.
       - `def get_llama_manager() -> LlamaServerManager | None:` — return _llama_manager.
       - Register all three hooks: `app.on_startup(_start_llama)`, `app.on_shutdown(_stop_llama)`, `app.on_disconnect(_stop_llama)`, `atexit.register(_stop_llama)`.

       **UI root (per D-03, D-14):**

               '/templates': templates.build,
               '/settings': settings.build,
           })
