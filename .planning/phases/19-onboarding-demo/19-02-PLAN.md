Создать новый файл app/demo_data.py.

Функция insert_demo_contracts(db) — вставляет демо-данные напрямую в SQLite без pipeline.

Данные должны демонстрировать ВСЕ возможности реестра:
- Разные типы: поставка, аренда, услуги, трудовой, подряд, NDA — не менее 6 типов
- Разные статусы: done, expiring (date_end = сегодня + 15 дней), attention
- Разные суммы (от 50 000 до 5 000 000 ₽)
- Реалистичные контрагенты (ООО «ИнноТех», ИП Смирнова А.В., ЗАО «Альфа» и т.д.)
- Некоторые с date_end в прошлом (expired), некоторые в будущем
- ai_confidence от 0.7 до 0.98
- processed_at — даты в диапазоне последних 6 месяцев

10 контрактов = достаточно чтобы stats bar показал числа, сегменты работали, calendar toggle имел события.

Реализация:
```python
import sqlite3
from datetime import date, timedelta
import hashlib

DEMO_CONTRACTS = [
    {
        "filename": "Договор_поставки_ИнноТех.pdf",
        "contract_type": "Договор поставки",
        "counterparty": "ООО «ИнноТех»",
        "date_start": "2025-01-15",
        "date_end": str(date.today() + timedelta(days=15)),  # istekает скоро
        "sum_rub": 1200000.0,
        "status": "done",
        "ai_confidence": 0.95,
        "processed_at": "2025-01-16 10:23:00",
        "source_path": "/demo/Договор_поставки_ИнноТех.pdf",
    },
    # ... ещё 9 контрактов
]

def insert_demo_contracts(db) -> int:
    """Вставляет demo контракты. Возвращает кол-во вставленных."""
    conn = db.conn
    inserted = 0
    for c in DEMO_CONTRACTS:
        file_hash = hashlib.md5(c["filename"].encode()).hexdigest()
        # Не дублировать если уже есть demo данные
        existing = conn.execute(
            "SELECT id FROM contracts WHERE file_hash = ?", (file_hash,)
        ).fetchone()
        if existing:
            continue
        conn.execute("""
            INSERT INTO contracts (
                filename, contract_type, counterparty, date_start, date_end,
                sum_rub, status, ai_confidence, processed_at, file_hash, source_path,
                is_latest_version, version_group_id
            ) VALUES (?, ?, ?, ?, ?, ?, ?, ?, ?, ?, ?, 1, NULL)
        """, (
            c["filename"], c["contract_type"], c["counterparty"],
            c["date_start"], c["date_end"], c["sum_rub"], c["status"],
            c["ai_confidence"], c["processed_at"], file_hash, c["source_path"],
        ))
        inserted += 1
    conn.commit()
    return inserted
```

Полный список 10 контрактов — разные типы, суммы, сроки. Минимум 2 с date_end в течение 30 дней (попадут в «Истекают»), минимум 1 с date_end в прошлом (статус "expired" или attention).
