Сделать ДО сборки десктоп-версий:

ИКОНКА: Отрендерить «Ю» в индиго в .icns (macOS) и .ico (Windows). Размеры: 16/32/128/256/512px. Без неё — стандартный питон-логотип.

SPLASH SCREEN: PyInstaller стартует 3-5 сек (распаковка). Нужен нативный splash или loading.html как первая страница чтобы юрист не видел пустоту.

OFFLINE FIRST RUN: Если нет интернета при первом запуске — GGUF не скачается, llama-server не стартует. Показать понятное "подключитесь к интернету для первой настройки" вместо крэша. Проверять connectivity перед скачиванием.

ПРОВЕРКА МЕСТА НА ДИСКЕ: Модели ~1.2GB + приложение ~400MB. Перед скачиванием проверить shutil.disk_usage() и предупредить если мало места.

ЛИЦЕНЗИЯ/EULA: Минимальное соглашение для установщика Windows (Inno Setup показывает при установке). "ПО предоставляется как есть, данные обрабатываются локально."
