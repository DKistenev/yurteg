Удалённые логи для бета-тестирования: BetterStack Logtail

ЗАЧЕМ: Отправляем .dmg друзьям для тестирования. Если что-то сломается — нужно видеть логи удалённо, не просить "скинь файл".

РЕШЕНИЕ: BetterStack Logtail (logtail.com). Бесплатно 1GB/мес — хватит на десятки тестеров.

ЧТО ДАЁТ:
- Live tail в браузере — видишь логи в реальном времени
- Поиск по тексту ошибок
- Фильтры по уровню (ERROR/WARNING), по тестеру, по дате
- Алерты в Telegram при массовых ошибках
- Dashboard со статистикой

РЕАЛИЗАЦИЯ: 5 строк в main.py — logging handler отправляет логи на HTTPS endpoint BetterStack. Каждый тестер идентифицируется через machine_id или имя (передаётся как extra field). Локальный файл-лог сохраняется параллельно (RotatingFileHandler).

БЕЗОПАСНОСТЬ: Логируем только имена файлов, статусы, ошибки. Содержимое юридических документов НЕ попадает в логи — это уже так в текущем коде.
