# Установка ЮрТэг

## Системные требования

- macOS 12 Monterey или новее (Apple Silicon)
- 4 ГБ свободного места на диске (приложение + языковая модель)

## Установка

1. Скачайте файл **YurTag-X.X.X.dmg** со страницы [Releases](https://github.com/danilakistenev/yurteg/releases)
2. Откройте скачанный DMG-файл двойным кликом
3. Перетащите **YurTag.app** в папку **Applications**

## Первый запуск (обход Gatekeeper)

Приложение не подписано сертификатом Apple, поэтому при первом запуске macOS заблокирует его. Это нормально -- нужно разрешить запуск один раз, после чего приложение будет открываться как обычно.

### Способ 1: Через Системные настройки (рекомендуется)

1. Откройте **YurTag.app** из папки Applications
2. macOS покажет диалог **"Не удается открыть приложение"** -- нажмите **OK**
3. Откройте **Системные настройки** > **Конфиденциальность и безопасность**
4. Прокрутите вниз до сообщения **"Приложение YurTag было заблокировано"**
5. Нажмите **"Все равно открыть"** и введите пароль администратора
6. При следующем запросе нажмите **"Открыть"**

### Способ 2: Через Терминал

Откройте приложение **Терминал** (Finder > Программы > Утилиты > Терминал) и выполните:

```bash
xattr -cr /Applications/YurTag.app
open /Applications/YurTag.app
```

Первая команда снимает карантинную метку, вторая запускает приложение.

## После установки

После первого успешного запуска приложение будет открываться как обычно -- повторный обход Gatekeeper не потребуется.

При первом запуске ЮрТэг автоматически скачает языковую модель QWEN (~1 ГБ). Убедитесь, что есть подключение к интернету.

## Удаление

1. Перетащите **YurTag.app** из папки Applications в Корзину
2. При желании удалите данные приложения:

```bash
rm -rf ~/.yurteg
```
